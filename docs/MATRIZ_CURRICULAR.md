## Límites éticos del módulo

- Solo se trabaja con autorización explícita.
- El alcance se define antes de cualquier prueba.
- Se utilizan datos ficticios y maquetas preparadas para aprender.
- No se escanean redes, servicios o dispositivos de terceros.
- No se solicitan credenciales reales ni se simula su recolección.
- No se enseña persistencia, evasión, distribución de malware ni explotación destructiva.
- Los hallazgos se comunican de forma privada, mínima y orientada a la corrección.

## Unidad piloto

**¿Confiarías en este mensaje?** integra análisis de phishing, ingeniería social, URL, urgencia, autenticación y verificación independiente. La evidencia principal es un protocolo de decisión y reporte aplicable a mensajes ficticios.
